## ⚠️ Note: When you want to start the game, make sure to select start.py and RUN the file and not debug.
For this variant of the exercise of the exchange program we do differently. Instead of dividing the game into sections we are going to make the full base game. Of course, you won't be doing it from scratch, but we have divided the game into three difficulties:

# Let's start with the Hard difficulty.
# You need the files joystick.py, start.py, and menu.py.

### What you need to fix
- Everything from the easy difficulty.
- Everything from the medium difficulty.
- You have to add a new feature to the game. You can add a new enemy type, a new power-up, or a new level. Be creative with it.
- You have to provide all your assets/spirits.

```
Tip: Always start with the ardiuno controler'(s) first, because if you don't have that working you won't be able to make the game that is intended to be played. 
You can always return to basics and make the game with keyboard controls, but the ardiuno controler is what makes this game special. So make sure to get that working first.
- See the ardiuno manual for more details.
```
