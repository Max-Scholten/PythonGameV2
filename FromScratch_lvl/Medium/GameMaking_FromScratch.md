# Let's start with the Medium difficulty.
# You need the files joystick.py, start.py, and menu.py.

### What you need to fix
- Everything from the easy difficulty.
- The detection of the COM port is not working, you have to fix that.
- The screen is a bit smaller than it should be, you have to fix that as well.
- Someone forgot to add a good ending to the game, you have to add that as well.
- The Menu screen is all white something is wrong with the code. You need to fix this because otherwise it will be hard to navigate through the game.
